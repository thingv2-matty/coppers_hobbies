# Brand vs Category Audit

Use this file to track which items appearing in the **Category** filter
are actually brands that should be moved to `KNOWN_BRANDS` in
`custom-code/search-engine-console-test.js` (and eventually `search-engine.html`).

---

## How to use this file

1. Open the search results page and expand the **Category** filter
2. If you spot a manufacturer/brand name sitting in Category instead of Brand,
   note it here under "Brands to add"
3. Once you have a batch ready, I'll add them to `KNOWN_BRANDS` in one go
